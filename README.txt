Программа MyNotebook представлена в двух вариантах:
Консольный - состоит из модулей model,presenter,view
Оконный - состоит из модулей model,tkinterview
Модуль model общий для обоих вариантов. Автор пытался следовать принципу Model View Presenter.




Работа с оконным вариантом
Открыть программу можно, запустив модуль tkinterview

1.Загрузка данных из файла - нажать кнопку "Загрузить записи"
2.Сохранение данных в файл - нажать кнопку "Сохранить записи"
3.Добавить новую запись - ввести название темы записи в поле "Тема\Дата", ввести сообщение в поле "Сообщение", нажать кнопку "Добавить новую запись"
4.Удалить запись - левый клик на нужной записи в поле ListBox затем нажать кнопку "Удалить запись". 
5.Редактировать запись - левый клик на нужной записи в поле ListBox. Внести изменения в тему и сообщение в соответствующих полях затем нажать кнопку "Редактировать запись".
6.Для поиска по теме - ввести фразу\слово\буквосочетание в поле "Тема\Дата" - нажать кнопку "Найти запись по теме"
7.Для поиска по дате - ввести дату согласно шаблону (например:2023,9,22) в поле "Тема\Дата" - нажать кнопку "Найти запись по дате"
8.Для поиска по сообщению - ввести фразу\слово\буквосочетание в поле "Сообщение" - нажать кнопку "Найти запись по сообщению"
Дата и время подтягиваются к созданной записи автоматически. Затем их уже нельзя отредактировать, кроме как отредактировать саму запись.


Работа с консольным вариантом
Открыть программу можно, запустив модуль presenter

1.Загрузить записи - нажать х и Enter
2.Сохранитть записи - нажать z и Enter
3.Удалить запись - нужно нажать s -> Enter, затем следовать инструкциям
4.Редактировать запись - нажать d -> Enter, затем следовать инструкциям
5.Чтоб осуществить поиск записи - жмем f->Enter затем следуем инструкциям.
